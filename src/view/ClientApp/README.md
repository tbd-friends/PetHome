# Pet Home Client

## Install

```
$ git clone git@github.com:tbd-friends/PetHome.git
$ cd .\PetHome\
$ git checkout -b feature/client_base_ts --track origin/feature/client_base_ts
$ cd .\src\view\ClientApp\
$ yarn
```

## Start & watch

```
$ yarn start
```

## Simple build for production

```
$ yarn build
```
